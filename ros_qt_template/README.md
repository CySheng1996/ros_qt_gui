# ROS_QT_GUI
## 說明
 1. 如果想添加新的程式，可以修改文件夹中的[Cmakelist.txt]
 2. 如果想添加新的圖形化控件，可以將檔案放入GUI中，並修改[Cmakelist.txt]
## 開發環境
 1. Ubuntu : 16.04
 2. Qt : 5.7.1 Qt creator : 4.2.1
 3. ROS : kinetic
